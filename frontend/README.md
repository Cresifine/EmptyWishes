# EmptyWishes Frontend

Flutter/Dart mobile app for EmptyWishes.

## Features

- Create and track wishes/challenges
- View progress updates
- Notifications
- User profile

## Setup

1. Ensure Flutter is installed:
```bash
flutter --version
```

2. Install dependencies:
```bash
flutter pub get
```

3. Run the app:
```bash
flutter run
```

## Build

```bash
# Android
flutter build apk

# iOS
flutter build ios
```
